## Usage

> For testing, nodemon has been installed as a dependency.  After installing dependecies, run the
following from the root directory
```sh
npm run build
npm start
```

## Requirements

- Node 7.10.0
- Webpack ^3.4.1

## Development

### Installing Dependencies

From within the root directory:

```sh
npm install
```
